# Design Specifications

## 1. Typography (Global)

### Font Family

* **Montserrat** (Canva-compatible)
* Use the same Montserrat family everywhere. If Canva shows multiple Montserrat variants, stay within the same family.

### Allowed Weights

* **Regular** – all body and base text
* **SemiBold** (or **Medium** if SemiBold is unavailable) – accents only
* **Italic** – optional, used exclusively for pronunciation guides

### Strict Rules

* Never use **Bold**
* Do not mix font families
* Accents must rely on weight and color, not size inflation

No additional colors allowed outside this palette.

## 5. Global Usage Rules

* Maximum **one accent** per slide
* Accent = **SemiBold only**, never Bold
* Pronunciation text must always be visually secondary
* One font system creates consistency and a high-end look
* Clarity > decoration
